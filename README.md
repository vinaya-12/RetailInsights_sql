 Retail Insights Data Analysis (SQL - Oracle)

 Description

Retail Insights is a SQL-based project that analyzes supermarket inventory and sales data to identify expiring products, dead stock, and revenue contribution by category.

---

 Features

* Relational schema design (3 tables)
* Inventory health analysis
* Dead stock identification
* Revenue aggregation

---
 Concepts Used

* SQL Joins
* GROUP BY & Aggregation
* Date Functions (SYSDATE, ADD_MONTHS)
* LEFT JOIN with NULL filtering

---

 Database Schema

* Categories
* Products
* SalesTransactions

---

 How to Run

Execute the `.sql` script in Oracle SQL Developer or SQL*Plus.

---

 Key Queries

 Expiring Soon

Find products expiring within 7 days with high stock.

 Dead Stock

Identify products with no sales in last 60 days.

 Revenue by Category

Calculate category-wise revenue for last month.

---

 Key Learning

Designed normalized schema and applied advanced SQL queries to generate business insights from transactional data.
